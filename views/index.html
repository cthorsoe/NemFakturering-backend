Welcome to the Team Up API

Available routes

Users

  • GET - /users/get-user/:username - Gets data of specific user where the passed username matches.
  • GET - /users/get-team-invites-count/:username - Gets the amount of curret team invites for the user where the passed username matches.
  • GET - /users/get-listed-users - Gets a list of users listed as looking for team.
  • GET - /users/get-my-teams/:username - Gets a list of teams which the user, which the passed username matches, is on.
  • POST - /users/create-user - Creates a new user based on the data passed in the body.
  • POST - /users/login - Tries to login based on username and password passed in the body.
  • POST - /users/login-by-session - Tries to login based on sessionSalt and sessionId passed in the body.
  • POST - /users/list-user - Updates data of user and lists the user as looking for team based on the data passed in the body.
  • PUT - /users/edit-user - Updates the data of an existing user based on the data passed in the body.
  • DELETE - /users/delete-user/:username - Deletes the user where the passed username matches.
  • DELETE - /users/delete-login-session/:id - Deletes the login session that matches the passed id.

Teams

  • GET - /teams/get-team/:id - Gets data of specific team, including members, where the passed id matches.
  • GET - /teams/get-listed-teams - Gets a list of teams listed as looking for players.
  • POST - /teams/create-team - Creates a new team based on the data passed in the body, adds the creator as the only member.
  • POST - /teams/list-user - Lists the team as looking for players where the id passed in the body matches.
  • PUT - /teams/edit-team - Updates the data of an existing team based on the data passed in the body.
  • DELETE - /teams/delete-team/:id - Deletes the user where the passed id matches.
